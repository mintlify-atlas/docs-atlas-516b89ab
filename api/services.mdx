## Overview

The Services API provides access to information about various services offered at Zoo Arcadia, including guided tours, restaurants, gift shops, and other visitor facilities.

## List All Services

<Note>
  Returns a complete list of all services and facilities available at the zoo.
</Note>

```http
GET /api/services
```

### Response Fields

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="array">
  Array of service objects
  
  <ResponseField name="id_service" type="integer">
    Unique identifier for the service
  </ResponseField>
  
  <ResponseField name="service_title" type="string">
    Title/name of the service
  </ResponseField>
  
  <ResponseField name="service_description" type="string">
    Detailed description of the service
  </ResponseField>
  
  <ResponseField name="link" type="string">
    URL or path related to the service (e.g., booking page)
  </ResponseField>
  
  <ResponseField name="type" type="string">
    Type of service: "featured", "service", or "habitat"
  </ResponseField>
  
  <ResponseField name="media_path" type="string">
    URL path to the service's image (original size)
  </ResponseField>
  
  <ResponseField name="media_path_medium" type="string">
    URL path to the service's medium-sized image
  </ResponseField>
  
  <ResponseField name="media_path_large" type="string">
    URL path to the service's large-sized image
  </ResponseField>
</ResponseField>

### Example Request

<CodeGroup>
```bash cURL
curl https://zoo-arcadia.example.com/api/services
```

```javascript JavaScript
fetch('https://zoo-arcadia.example.com/api/services')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));
```

```python Python
import requests

response = requests.get('https://zoo-arcadia.example.com/api/services')
services = response.json()
print(services)
```
</CodeGroup>

## Service Types

Services are categorized into three types:

### Featured Services

Featured services are highlighted offerings displayed prominently on the homepage. These typically include:
- Special tours
- Premium experiences
- Educational programs

### Regular Services

Standard services and facilities available to all visitors:
- Restaurants and cafes
- Gift shops
- Restrooms and facilities
- Information centers

### Habitat Services

Services specific to individual habitats:
- Habitat-specific guided tours
- Feeding demonstrations
- Educational presentations

## Error Responses

### Resource Not Found

When the services endpoint is not available:

```json
{
  "error": true,
  "message": "Resource 'services' not found"
}
```

**HTTP Status Code:** `404 Not Found`

## Notes

<Note>
  The Services API returns all services regardless of type. You can filter services by type on the client side using the `type` field.
</Note>

<Note>
  Service images are available in three sizes (original, medium, large) to optimize for different display contexts and screen sizes.
</Note>
